## Business question routing

**"Why no leads?"** → Distribution (posting?), conversion (offer/CTA?), follow-up (DMs?). Hormozi value equation + Brunson HSO on last 5 posts.

**"What should I post?"** → Invisibility Diagnostic + ICP night's thought. Content engine, not random ideas.

**"How do I price?"** → Hormozi stack + filter logic. Stage-aware (early proof vs established).

**"Workshop not converting?"** → Brunson Perfect Webinar skeleton + Taki cadence + buyer flagging in first 48h.

**"Which AI tools?"** → Claude Projects + skills + brain index. Model is commodity; **system** is moat.

## AI + marketing combined

The AI Marketing Employee pattern:
1. Voice Profile (4 inputs)
2. ICP doc locked
3. Skills for post / carousel / cheatsheet / outreach
4. Weekly batch, human ships

Experts inform **what** to say (offer, hook, story). AI handles **draft speed**. Founder owns **voice and POV**.

## Regular updates

External repo sync (`npm run sync-experts`) pulls team KB from GitHub weekly. Vault sync (`npm run sync-knowledge`) pulls Obsidian. Both re-index into brain.

When expert repos update, answers get sharper without retraining the model.
